    Rubik's cube Tutorial

    Rubik's cube Tutorial

    Third layer

    Awesome~! Now you are only one step left to solve the cube.

    First case

    You can't find corner colour is same (third layer) in any sides. Please apply "RB'RF'F'R'BRF'F'RR". After you applied, you should see same corner colour in one side or more sides.

    Second case

    You can find corner colours are same (third layer) in one sides. Please apply "RB'RF'F'R'BRF'F'RR". After you applied, you should see all the corners are same colour.

    Third case (a)

    all centre blocks are different in third layer, after you apply "R2UF'R2F'BUR2". You should see only three.

    Third case (b)

    three of the centre blocks are different, please use the formula "R2UF'R2F'BUR2", you need to be awared the scenario. If your scenario is inversed, you only need to apply this formula two times. And the Rubik's cube should be solved by you~! Congratulations.
