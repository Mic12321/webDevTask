    Rubik's cube Tutorial

    Rubik's cube Tutorial

    Top

    Now we are going to solve the top~! Let's assume you solved the first side colour is white. Then, the top will be yellow. The first step is to create the top cross shape. And the final step is to make all the top colours to be same. There are so many cases. For beginners, we only need to use some formulas to solve it. If you want to be faster, you need to learn and practice a lot of formulas.

    First case

    You can't find two or more top colour edges on top side. Please apply "R'U'F'UFR". After you finished, you should see two edges are on the top side.

    First case result

    If you can see there are two top colour edges are on top side. Congratulations, you applied a correct formula.

    Second case (a)

    You can find two or more top colour edges on top side. Please apply "R'F'U'FUR". If you can see one of the third cases, you applied correct.

    Second case (b)

    You can find two or more top colour edges on top side. Please apply "R'UF'UFR". If you can see one of the third cases, you applied correct.

    Thrid case (a)

    You can find two or more top colour edges on top side. Please apply "RUR'URU'U'RU'". After you finished, you should see the top are all the same color.

    Thrid case (b)

    You can find two or more top colour edges on top side. Please apply "R'U'RU'R'UURU". After you finished, you should see the top are all the same color.

    Thrid case (c)

    You can find two or more top colour edges on top side. Please apply "R'R'D'RUUR'DRUUR". After you finished, you should see the top are all the same color.

    Thrid case (d)

    You can find two or more top colour edges on top side. Please apply "RUR'URU'R'URU'U'R'". After you finished, you should see the top are all the same color.

    Thrid case (e)

    You can find two or more top colour edges on top side. Please apply "B'R'F'RBR'FR". After you finished, you should see the top are all the same color.

    Thrid case (f)

    You can find two or more top colour edges on top side. Please apply "R'F'RB'R'FRB". After you finished, you should see the top are all the same color.

    Thrid case result

    If you see all the top colours are same. Congratulations, you can go to next step.
