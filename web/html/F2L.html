    Rubik's cube Tutorial

    Rubik's cube Tutorial

    Cross

    The first step is to make one of the side to have the cross shape. For beginners, colour white is suggested to practice. Please remember the cell in the middle of the top row need to be the same colour as the center cell.

    First Layer Corner

    The second step is to make the edge to be correct. Please remember the edge blocks need to be matced the colour with different sides.

    Second Layer

    Well done~! Now you have solved one side with one layer. Now, we will apply some formulas to solve it quickly. Please flip your cube. The fully solved side will be the side of bottom. And there are two cases to solve the second layer.

    First case

    Please apply the formula "URU'R' U'F'UF". If you apply the correct formula in correct scenario, the block colour will be matched both sides.

    First case result

    If the colour does not match, you may applied the wrong formula to the case.

    Second case

    Please apply the formula "U'F'UF URU'R'". If you apply the correct formula in correct scenario, the block colour will be matched both sides.

    Second case result

    If the colour does not match, you may applied the wrong formula to the case.

    The wrong result or some cases

    In this case, you can see the colour is flipped. You can either apply one of the formulas that mentioned above. Then, the block does not match colour will be released. After that, you can solve it and see the scenario above.

    The first two layer result

    After you finished, you can check is it same as the picture (First two layers only). If it is same, congratulations. You can go to the next step.
